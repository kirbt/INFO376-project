# Sample Files for DocuSearch

This folder contains sample files to test the DocuSearch indexing and search capabilities.

## Supported Formats:
- **.pdf**: Full PDF parsing and indexing.
- **.txt**: Plain text indexing.

## Contents:
- `ai_future.txt`: Information about Artificial Intelligence.
- `climate_change.txt`: Discussion on climate change and renewable energy.
- `space_exploration.txt`: Overview of modern space exploration.

You can upload these files individually or all at once using the "Upload Documents" button in the web UI.
